# Turn-Based RPG Game (WIP)

A console-based turn-based RPG game built in C++.  
Currently ~70% complete. Designed to explore pointers, OOP, and core C++ functionality.

---

## Features Implemented

- **Turn-based combat** between player and enemy characters  
- **Character stats:** health, attack, defense, and level  
- **Item system:** simple inventory with healing items  
- **Basic enemy AI:** enemies take turns attacking  
- **Interactive console interface** for choosing actions  

---

## Planned Features

- **Save/Load system** for player progress
- Expanded **enemy types, abilities and levels**  
- **Skills and skill tree system**  
- **More complex inventory and equipment system**  
- Full **save/load functionality**
- Polished **combat UI and feedback messages**
- Basic graphical addition (later)
